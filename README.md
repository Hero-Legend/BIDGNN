# BIDGNN: BioGPT-Enhanced Dual-Graph Framework for DDI Extraction

This repository contains the official implementation and data for the paper: **"Unveiling Rare Drug Interactions: A BioGPT-Enhanced Dual-Graph Framework for Robust Pharmacovigilance"**.

BIDGNN synergizes generative augmentation (BioGPT-GAN) with structural reasoning (Dual-Graph Neural Network) to solve the "long-tail" and "semantic ambiguity" problems in DDI extraction.

## üìÇ Project Structure

- **`maincode_DGNN.py`**: The core script for the Dual-Graph Neural Network (DGNN) model. It handles data loading, graph construction, training, and evaluation.
- **`Data Augmentation.py`**: The BioGPT-GAN module used to generate synthetic samples for rare classes (Advice, Int).
- **`train.tsv` / `dev.tsv` / `test.tsv`**: The original DDI 2013 dataset splits.
- **`augmented_train_data.tsv`**: The augmented training dataset generated by our BioGPT-GAN.
